Features

Everything You Need

Built with security, performance, and developer experience in mind.

Instant Generation

Generate secure numeric or alphanumeric OTPs instantly. Configurable length and complexity.

Cryptographically Secure

Uses strong randomization for OTP generation with automatic expiration and one-time use enforcement.

Email Delivery

Automatic email delivery via SMTP with customizable organization name and email subject.

Auto-Expiration

OTPs automatically expire after a configurable duration. Defaults to 2 minutes for maximum security.

Spam Protection

Built-in IP and email blocklist with keyword filtering. Prevents abuse automatically.

Easy Integration

Simple REST API with JSON payloads. No API key required — just send a POST request and go.

API Reference

API Endpoints

Two simple endpoints — generate and verify. That's all you need.

No API Key Required

This service is completely free and open. Just send your requests — no authentication, no sign-up, no rate limits beyond spam protection.

POST

Generate & Send OTP

Generate a new OTP and deliver it to the specified email address.

URL /api/otp/generate
Request Body (JSON)
email string required Recipient's email address
type string optional "numeric" (default) or "alphanumeric"
organization string optional Organization name shown in the email
subject string optional Custom email subject line
Response 200 OK
{
  "message": "OTP is generated and sent to your email"
}
POST

Verify OTP

Verify the OTP entered by the user against the stored value.

URL /api/otp/verify
Request Body (JSON)
email string required The email address used to generate the OTP
otp string required The OTP code to verify
Response 200 OK
{
  "message": "OTP is verified"
}
Quick Start

Integrate in Seconds

Copy, paste, and you're ready to go. No SDK, no setup.

generate-otp.js 
// Step 1: Generate & send OTP
const response = await fetch('https://otpservice-free.vercel.app/api/otp/generate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'user@example.com',
    organization: 'My App',
    subject: 'Verify Your Account'
  })
});

const data = await response.json();
console.log(data);
// → { "message": "OTP is generated and sent to your email" }

// Step 2: Verify OTP (after user enters the code)
const verify = await fetch('https://otpservice-free.vercel.app/api/otp/verify', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'user@example.com',
    otp: '482916'
  })
});

const result = await verify.json();
console.log(result);
// → { "message": "OTP is verified" }
